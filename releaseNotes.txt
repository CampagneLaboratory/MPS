New Features in MPS 3.0

- Tight integration between Java and BaseLanguage in the MPS plugin for IntelliJ IDEA
- Import from Java
- BaseLanguage structure and editor changes
- Face-lifted Structure language
- Open API to access model repositories
- Custom persistence API
- Build language improvements
- Local history
- Version control improvements
- Default editor and multiple editors per concept
- Overriding editor components
- Type system changes and type system debugging improvements

See more on MPS web site: http://www.jetbrains.com/mps/whatsnew/index.html
